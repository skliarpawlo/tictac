#### Simple metronome

 Have only feature yet: acceleration
 
 
#### Usage

 tictoc.py --bpm 60 --mins 0.1 speed-up 10 --upper-limit 70
 

#### Install

 - install pyAudio http://people.csail.mit.edu/hubert/pyaudio/
 - install wave
 - git clone this repo
 
 
#### Test

 - install nose
 - from root directory run: nosetests
